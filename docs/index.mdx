# iQuest Agentic Test Harness

**Discover. Validate. Assure.**

The open-source agentic testing harness for browser and API automation. Write your tests in plain English. Run `npx iquest init`. Done.

<Note>
  An **agent harness** (coined by [Martin Fowler](https://martinfowler.com/articles/harness-engineering.html) and described by [Anthropic](https://www.anthropic.com/engineering/managed-agents)) is the infrastructure that wraps an AI model — connecting it to tools, managing its context, and steering its behaviour. iQuest is that harness, purpose-built for browser and API testing. Built for **agentic engineering** workflows — the same agent that builds your product can verify it.
</Note>

## All You Need to Write

A `.feature` file:

```gherkin
Feature: TodoMVC Automation

  Scenario: Add todo item
    * Navigate to "https://demo.playwright.dev/todomvc/"
    * Add a new todo item "Buy groceries"
    * Should see "Buy groceries" in the todo list
```

Then run:

```bash
npx iquest init   # one-time setup
cd .iquest
npm run test:headed
```

That's it. No step definitions. No selectors. No code.
